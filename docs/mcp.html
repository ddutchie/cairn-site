MCP Server

Cairn ships a bundled stdio MCP server. Connect Claude Desktop, OpenCode, Cursor, or any MCP-compatible client and give it direct read/write access to your workspace. Changes appear in the Cairn UI instantly.

What is MCP?

Model Context Protocol (MCP) is an open standard published by Anthropic for exposing structured tools to AI clients. An MCP server runs as a local process alongside your AI client; the client calls tools from the server to read and write data. Cairn's MCP server exposes your entire workspace — notes, tasks, projects — to any compatible client.

Where the server binary lives

The MCP server is bundled inside the installed Cairn app. It does not require a separate install.

macOS (installed app)

~/Applications/Cairn.app/Contents/Resources/app.asar.unpacked/dist-mcp/mcp-server.bundle.js

macOS (running from source)

/path/to/cairn/dist-mcp/mcp-server.bundle.js

The native SQLite binding lives next to the bundle:

dist-mcp/../mcp-native/better_sqlite3_node.node

Configuring Claude Desktop

Add the following to ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "cairn": {
      "command": "node",
      "args": [
        "/Users/YOUR_USERNAME/Applications/Cairn.app/Contents/Resources/app.asar.unpacked/dist-mcp/mcp-server.bundle.js"
      ]
    }
  }
}

Replace YOUR_USERNAME with your macOS username. Restart Claude Desktop after saving.

Configuring OpenCode

In your project repo, create .opencode/config.json:

{
  "mcp": {
    "cairn": {
      "type": "stdio",
      "command": "node",
      "args": [
        "/Users/YOUR_USERNAME/Applications/Cairn.app/Contents/Resources/app.asar.unpacked/dist-mcp/mcp-server.bundle.js"
      ]
    }
  }
}

How writes appear instantly

The Cairn app polls the SQLite database using WAL (Write-Ahead Logging) change detection. When the MCP server writes a note or task, the change is committed to the database and the app detects it within milliseconds — no restart required, no manual refresh. The UI updates in place.

Tool reference

All 15 tools exposed by the Cairn MCP server:

Tool Read / Write Description
get_cairn_context Read Returns all workspaces and projects with column IDs and tool conventions
get_project_summary Read Card counts by column, pinned notes, and recent activity for a project
list_recent_activity Read Recently created or updated notes and tasks in a workspace or project
search_notes Read Full-text search across notes, optionally filtered by project
get_note Read Returns full markdown content of a note by ID
create_note Write Creates a new note in a project
update_note Write Updates a note's title or markdown content
delete_note Write Permanently deletes a note
search_tasks Read Searches task cards by text, project, or column type
get_task Read Returns full detail of a task card including linked notes
create_task Write Creates a task card in a specified column
update_task_status Write Moves a task card to a different column
delete_task Write Permanently deletes a task card
link_note_to_task Write Creates a bidirectional link between a note and a task card
create_project Write Creates a new project with default board columns
IDs are stable

Note and task IDs are stable across sessions. Use get_cairn_context at the start of a session to get current project and column IDs, then reuse them for subsequent tool calls.
